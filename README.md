# PacMan

My second assignment with C# and windows forms. To download the latest version, check releases tab.

Controls:

[R]         - Restart the game

[Space]     - Shoot if possible

[Q]         - Quit the game

[ArrowKeys] - Control the pacman

[X]         - Debug mode

[Enter]     - Start the game
